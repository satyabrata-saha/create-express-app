# create-express-app

create express app with folder structure

# steps

```bash
mkdir app-name
```

```bash
cd app-name
```

```bash
npm init -y
```

```bash
npm i express ejs body-parser dotenv
```

```bash
touch .env .gitignore Readme.md
```

```bash
mkdir public src
```

```bash
cd src
```

```bash
mkdir controllers db middlewares models routes utils
```

```bash
touch app.js constant.js index.js
```
